\begin{abstract}
Questo documento descrive l'architettura e l'implementazione di un motore di scacchi
scritto in C++ moderno (C++23). Il progetto implementa un'intelligenza artificiale
capace di giocare a scacchi utilizzando bitboards, alpha-beta pruning, transposition
tables e numerose ottimizzazioni moderne.
\end{abstract}
